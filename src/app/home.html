This workflow serves as a starting point for building AngularJS (1.x) with AngularRoutes and AngularAnimate applications using Webpack 2.x. and Bootstrap 4.x Should be noted that apart from the pre-installed angular package, this workflow is pretty much generic.

  1. Heavily commented webpack configuration with reasonable defaults.
  2. Angular-Routes included
  3. ES6, and ES7 support with babel.
  4. Source maps included in all builds.
  5. Development server with live reload.
  6. Production builds with cache busting.
  7. Testing environment using karma to run tests and jasmine as the framework.
  8. Code coverage when tests are run.
  9. No gulp and no grunt, just npm scripts.

Quick start

# clone our repo
$ git clone https://github.com/raphaelSm/angularjs-webpack-bootstrap.git 

# change directory to your app
$ cd my-app

# install the dependencies with npm
$ npm install

# start the server
$ npm start
